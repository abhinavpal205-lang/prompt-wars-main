# Sahaay — supportive wellbeing check-ins for exam-stage students

[![CI](https://github.com/OWNER/REPO/actions/workflows/ci.yml/badge.svg)](../../actions/workflows/ci.yml)

A gentle, non-clinical wellbeing check-in companion for students preparing for
high-pressure Indian exams (NEET, JEE, CUET, CAT, GATE, UPSC, boards).

> **Sahaay is not a diagnostic or clinical tool.** It is a self-reflection
> aid. If you or someone you know is struggling, call **Tele-MANAS:
> 14416 or 1-800-891-4416** (free, 24×7, India).

_Build in progress — full docs land with the final step._

## Quick start

```bash
cp .env.example .env   # optionally fill OPENAI_API_KEY / SMTP_*
docker compose up --build
# open http://localhost:3000
```
